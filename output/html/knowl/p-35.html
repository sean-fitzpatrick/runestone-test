Division doesn't quite make sense in this context, but it certainly does make sense to multiply both sides by \(1/c\text{,}\) the multiplicative inverse of \(c\text{.}\) We then have \((1/c)(c\vv)=(1/c)\ww\text{,}\) and since scalar multiplication is associative, \((1/c\cdot c)\vv = (1/c)\ww\text{.}\) We know that \(1/c \cdot c = 1\text{,}\) so this boils down to \(1\vv=(1/c)\ww\text{.}\) It appears that we've solved the equation, but only if we know that \(1\vv = \vv\text{.}\)
