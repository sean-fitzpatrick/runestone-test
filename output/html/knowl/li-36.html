Item 5.

The set \(F[a,b]\) of all functions \(f:[a,b]\to \R\text{,}\) where we define \((f+g)(x)=f(x)+g(x)\) and \((cf)(x)=c(f(x))\text{.}\) The zero function is the function satisfying \(0(x)=0\) for all \(x\in [a,b]\text{,}\) and the negative of a function \(f\) is given by \((-f)(x)=-f(x)\) for all \(x\in [a,b]\text{.}\)

Note that while the vector space \(P(\R)\) has an infinite nature that \(P_n(\R)\) does not, the vector space \(F[a,b]\) is somehow more infinite! Using the language of SectionÂ 1.7, we can say that \(P_n(\R)\) is finite dimensional, while \(P(\R)\) and \(F[a,b]\) are infinite dimensional. In a more advanced course, one might make a further distinction: the dimension of \(P(\R)\) is countably infinite, while the dimension of \(F[a,b]\) is uncountable.
