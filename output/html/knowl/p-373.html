Much of this theorem sums up some of what we've learned so far: As long as a vector space \(V\) contains a nonzero vector \(\vv\text{,}\) the set \(\{\vv\}\) is independent and can be enlarged to a basis, by Lemma 1.7.12. The size of any spanning set is at least as big as the dimension of \(V\text{,}\) by Theorem 1.7.1.
