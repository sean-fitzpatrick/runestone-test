Item 2.2.17.1.

Suppose \(T:V\to W\) is injective. Then \(\ker T = \{0\}\text{,}\) so

\begin{equation*} \dim V = 0 + \dim \im T \leq \dim W\text{,} \end{equation*}

since \(\im T\) is a subspace of \(W\text{.}\)

Conversely, suppose \(\dim V\leq \dim W\text{.}\) Choose a basis \(\{\vv_1,\ldots, \vv_m\}\) of \(V\text{,}\) and a basis \(\{\ww_1,\ldots, \ww_n\}\) of \(W\text{,}\) where \(m\leq n\text{.}\) By TheoremÂ 2.1.8, there exists a linear transformation \(T:V\to W\) with \(T(\vv_i)=\ww_i\) for \(i=1,\ldots, m\text{.}\) (The main point here is that we run out of basis vectors for \(V\) before we run out of basis vectors for \(W\text{.}\)) This map is injective: if \(T(\vv)=\zer\text{,}\) write \(\vv=c_1\vv_1+\cdots + c_m\vv_m\text{.}\) Then

\begin{align*} \zer \amp = T(\vv)\\ \amp = T(c_1\vv_1+\cdots + c_m\vv_m)\\ \amp = c_1T(\vv_1)+\cdots + c_mT(\vv_m)\\ \amp = c_1\ww_1+\cdots +c_m\ww_m\text{.} \end{align*}

Since \(\{\ww_1,\ldots, \ww_m\}\) is a subset of a basis, it's independent. Therefore, the scalars \(c_i\) must all be zero, and therefore \(\vv=\zer\text{.}\)
