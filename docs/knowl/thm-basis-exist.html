Theorem 1.7.13.

Any finite-dimensional (non-trivial) vector space \(V\) has a basis. Moreover:

  1. If \(V\) can be spanned by \(m\) vectors, then \(\dim V\leq m\text{.}\)

  2. Given an independent set \(I\) in \(V\) that does not span \(V\text{,}\) and a basis \(\mathcal{B}\) of \(V\text{,}\) we can enlarge \(I\) to a basis of \(V\) by adding elements of \(\mathcal{B}\text{.}\)
