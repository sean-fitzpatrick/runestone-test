Solution 1.8.12.a.1.

Since \(p(1)=0\text{,}\) we know that \(p(x)=(x-1)q(x)\text{,}\) for some \(q(x)=ax^2+bx+c\text{.}\) Thus, \(p(x)=ax^2(x-1)+bx(x-1)+c(x-1)\text{,}\) so a basis is given by \(\{(x-1),x(x-1),x^2(x-1)\}\text{.}\)

(Another option is \(\{(x-1),(x-1)^2,(x-1)^3\}\text{.}\))
