Let \(x\) be any element fo \(V\text{.}\) Since \(x\gt 0\text{,}\) we know in particular that \(x\neq 0\text{,}\) so we can define \(-x = 1/x\text{,}\) where \(1/x\) denotes the usual reciprocal of a real number. We then have

\begin{equation*} x\oplus (-x) = x(1/x) = 1\text{,} \end{equation*}

and we saw above that \(1\) is the identity element of \(e\text{.}\)
