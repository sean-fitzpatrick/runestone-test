For any operator \(T:V\to V\text{,}\) there are four subspaces that are always \(T\)-invariant:

\begin{equation*} \{\zer\}, V, \ker T, \text{ and } \im T\text{.} \end{equation*}

Of course, some of these subspaces might be the same; for example, if \(T\) is invertible, then \(\ker T = \{\zer\}\) and \(\im T = V\text{.}\) (We will skip the proof that \(\ker T\) and \(\im T\) are \(T\)-invariant, since this has been assigned as homework!)
